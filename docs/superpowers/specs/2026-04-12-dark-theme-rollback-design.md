# 深色主题回滚设计

日期：2026-04-12

## 目标

将当前智能分析工作台的深色主题从新版蓝色科技风回滚到之前的黑灰底 + 琥珀强调色方案，同时保留已经完成的布局、抽屉预览、项目总览卡片层级和快捷操作区位置调整。

## 本次回滚范围

- 仅调整 [frontend/app/globals.css](/D:/code/nanobot/frontend/app/globals.css) 中深色主题变量。
- 回滚 `--surface-0` 到 `--surface-3`、`--canvas-rail`、`--paper-chat`、`--paper-card`、`--text-primary`、`--text-secondary`、`--text-muted`、`--accent`、`--accent-soft`。
- 保持浅色主题和护眼主题不变。
- 保持布局、抽屉逻辑、Project Overview 卡片结构、空状态组件、快捷操作条位置不变。

## 预期结果

- 整体视觉恢复到之前熟悉的黑灰体系。
- 按钮、胶囊、进度高亮重新使用琥珀色强调。
- 不影响当前已经完成的交互结构优化。

## 风险与校验

- 风险较低，主要是主题变量回退后局部组件可能出现层级对比不足。
- 校验方式以浏览器实际刷新查看为主，重点检查聊天区、项目总览卡片、预览抽屉、快捷操作胶囊和大盘高亮态。
